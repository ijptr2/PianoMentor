System Overview

This prototype demonstrates a dual-device piano learning system with AI-powered suggestions and feedback. The system consists of three main components:

  1. Mobile Piano App - A Flutter mobile application that serves as the piano keyboard interface
  2. Web Dashboard - A Flutter web application that displays real-time feedback and visualizations
  3. Backend Server - This Flask server that processes data, generates AI suggestions, and handles communication between components

API Endpoints

Get Real-time Notes
Get the most recent notes being played
GET /api/notes
Save Note
Save a played note to the system
POST /api/save-note
Get Sessions
Get all practice sessions
GET /api/sessions
Get AI Suggestions
Generate AI practice suggestions based on played notes
POST /api/suggestions
Analyze Scale
Analyze what scale is being played
POST /api/analyze-scale
Get Daily Goal
Get the daily practice goal
GET /api/daily-goal
Get Progress Report
Generate a progress report for practice sessions
GET /api/progress-report

Testing the API

You can test the API endpoints using curl or other HTTP clients:

curl http://localhost:5000/api/notes

Or send a note to the system:

curl -X POST -H "Content-Type: application/json" \
-d '{"note": {"midiNote": 60, "velocity": 100, "timestamp": 1623456789, "isNoteOn": true}, "sessionId": "test-session"}' \
http://localhost:5000/api/save-note
